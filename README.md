# YearPlanner

YearPlanner. Application to plan your year with ease with whole year overview and several presentations of the year.

## Tech Stack

- React - view
- TailwindCSS and DaisyUI - UI
- Nanostores - state
- Typescript, Vite

## Getting Started

### Prerequisites

- Node.js (v20 or higher)
- pnpm, npm or yarn
